Folha de estilos em cascata - Cascading Style Sheets

Definição

-Folha de estilos em cascata - Cascading Style Sheets,usada para aplicar formatação e estilos às páginas html, controla a apresentação dos documentos, é uma coleção de regras de formatação que será aplicado aos html, pode ser aplicado das seguintes maneiras:

  • Inline: estilos usados diretamente em um elemento html;
  • Incorporado: O elemento style permite aplicar várias regras de uma vez no
    • próprio documento html;
  • Folha de estilos externa: um documento separado com a extensão .css, que permite aplicar as regras em um site de uma só vez.

Observação: Os estilos são aplicados em cascata, se um estilos é aplicado sobre um elemento e posteriormente outro estilo sobre o mesmo elemento, contará a última aplicação.

Sintaxe CSS

  • Seletor: Quem eu vou aplicar a regra, {"corpo do seletor"};
  • Propriedade: O que eu vou modificar no seletor;
  • Valor: Valor a ser aplicado nesta propriedade

Tipos de Seletores:

  • *: pega todos os elementos do documento e configura o valor eles;
  • por tag: pega todas a tags e seta o valor para elas;
  • Seletores por id: pega o id do elemento no html e modifica o mesmo utilizando “#” para acessá-lo no documento de css.
  • seletores por class: pega vários elementos do html e o modifica. Utiliza “.” para acessá-lo no documento de css.

Cuidados para class e id:

  • Não usar espaços em branco;
  • Não empregar caracteres especiais;
  • css é case-sensitive.
