No html, são:

<nav> //representa uma seção de uma página que aponta para outras páginas ou para outras áreas da página, no caso o menu principal

<ul> //Lista não ordenada

<li> // Item da lista, no caso do menu

No CSS, temos:



Estrutura body{margin=0;} //atribui a margem para 0

Estrutura ul{ list-style: none; padding: 0; overflow: hidden; transition: all .5s ; } //atribui para todas as lista ul os valores passados

h1, p{ color: #fff; } //Atribui aos titulos h1 e todos os parágrafos, a cor desejada

nav, ul { display: flex; flex-direction: row; } //atribui ao nav e ul os atributos de alinhamento desejados

li{ padding: 0 20px; } //Ajusta o padding

h1{ margin:0; padding: 10px; }//ajusta o padding e a margem

button{ background-color: transparent; border: none; color: #fff; font-size: 36px; position: absolute; right: 10px; top: 10px; display: none; }// No botão é atribuido cor de fundo, borda,cor, tamanho da fonte, posição e alinhamentos.

li:hover, h1:hover,button:hover{ cursor: pointer; opacity: 0.8; background-color: rgba(255, 255, 255, 0.2); }//ajusta a animação hover para o botão do menu, a cor que fica quando animado e opaciadade

button:focus ~ul{ height: 260px; }//Ajusta o height quando clicado

@media (max-width: 700px){ //nesta linha, de acordo com o tamanho em px, é aplicado as alterações as tags contidas dentro do mesmo nav, ul{ flex-direction: column; }//muda a orientação do menu

ul{ text-align: center; margin: 0; height: 0; }//alinha a lista do menu

nav{ padding-top: 0px; align-items: flex-start; }//alinha o menu

h1{ margin: 10px; }//ajusta a margem

button{ display: flex; }//alinha o botão

li, ul{ width:100%; }//ajusta a largura dos itens e da lista menu }
